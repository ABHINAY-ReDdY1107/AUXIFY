Website redesign review

Premium agency portfolio system for Auxify.

The redesigned website preserves the existing luxury palette and type direction while rebuilding the experience into a cleaner, more cinematic, and more professional digital studio presence.

DeliverableStatic production website
FocusPortfolio, services, motion, contact
PriorityAnimated wallet project showcase

Design decisions

From student agency template to premium studio language.

Visual system

  • Kept the gold, cream, and dark charcoal brand palette.
  • Preserved the existing font families: Cormorant Garamond and DM Sans.
  • Introduced larger editorial headings, quieter body copy, and more deliberate section spacing.
  • Removed cluttered elements, emoji-heavy project visuals, and inconsistent inline styling.

Brand expression

  • Positioned Auxify as a lean AI-powered digital studio.
  • Shifted copy from casual student phrasing to professional agency language.
  • Used subtle luxury materials: grain, fine lines, soft shadows, and gold accents.
  • Created original project mockup artwork for each featured build.

Animation system

Premium motion with restraint.

Implemented motion

  • Page-load brand sequence with a controlled reveal.
  • Hero text line reveal using GSAP when available.
  • ScrollTrigger parallax in the hero and wallet showcase.
  • Staggered section reveals via IntersectionObserver fallback.
  • Animated image masking on project case-study visuals.

Micro interactions

  • Magnetic buttons and brand link interactions on pointer devices.
  • Smooth custom cursor ring with hover expansion.
  • Service card elevation and animated bottom borders.
  • Project card hover depth, shadow, and inspection states.
  • Reduced-motion handling for accessibility.

Project showcase

The wallet becomes the centerpiece.

The previous stacked project area was rebuilt into a sticky scroll scene. A realistic dark leather-style wallet anchors the section, while project cards slide outward in sequence with depth, rotation, layering, and premium shadows. The cards behave like collectible design objects and are followed by full project storytelling panels.

Content changes

Resume information was used selectively.

The resume was reviewed and used only where it strengthened credibility. CGPA, school percentages, sports awards, and unrelated academic details were intentionally excluded. The site now references relevant project history and practical capabilities: Python, SQL, Data Analysis, Figma, Framer, N8N, AI tools, automation thinking, and product/interface work.

Responsiveness and accessibility

Built to feel finished across screen sizes.

Responsiveness

  • Mobile navigation with accessible expanded state.
  • One-column mobile layouts for services, about, contact, and case studies.
  • Wallet animation falls back to stacked project cards on small screens.
  • Stable dimensions for cards, wallet, timeline, and form controls.

Accessibility

  • Semantic sections, headings, forms, labels, and navigation.
  • Skip link for keyboard users.
  • Reduced-motion media query and animation fallback behavior.
  • Decorative images hidden from assistive technology where appropriate.
  • Improved color contrast across dark and light sections.

Technical cleanup

Cleaner structure and production-ready organization.

  • Replaced the large single-file website with organized HTML, CSS, JavaScript, and asset files.
  • Removed duplicate inline scripts and inconsistent inline CSS patterns.
  • Added graceful fallbacks if GSAP or ScrollTrigger fail to load.
  • Kept the site static and deployable from the Auxify folder.
  • Created local SVG mockups to avoid broken image dependencies.

Final files are located in the AUXIFY folder on the desktop.
